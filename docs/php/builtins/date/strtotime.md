## strtotime()

```php
function strtotime(string $datetime, int $baseTimestamp = null): mixed
```

Parses an English textual datetime description into a Unix timestamp.

**Parameters**:
- `$datetime` (`string`)
- `$baseTimestamp` (`int`), default `null`, optional

**Returns**: `mixed`
