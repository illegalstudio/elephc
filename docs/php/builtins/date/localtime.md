## localtime()

```php
function localtime(int $timestamp = -1, bool $associative = false): array
```

Returns the local time.

**Parameters**:
- `$timestamp` (`int`), default `-1`, optional
- `$associative` (`bool`), default `false`, optional

**Returns**: `array`
