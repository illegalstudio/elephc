## fopen()

```php
function fopen(string $filename, string $mode, bool $use_include_path = false, mixed $context = null): mixed
```

Opens file or URL.

**Parameters**:
- `$filename` (`string`)
- `$mode` (`string`)
- `$use_include_path` (`bool`), default `false`, optional
- `$context` (`mixed`), default `null`, optional

**Returns**: `mixed`
