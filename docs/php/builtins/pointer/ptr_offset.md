## ptr_offset()

```php
function ptr_offset(pointer $pointer, int $offset): mixed
```

Returns a new pointer offset from the given pointer by the given byte count.

**Parameters**:
- `$pointer` (`pointer`)
- `$offset` (`int`)

**Returns**: `mixed`
