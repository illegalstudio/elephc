## in_array()

```php
function in_array(mixed $needle, array $haystack, bool $strict = false): mixed
```

Checks if a value exists in an array.

**Parameters**:
- `$needle` (`mixed`)
- `$haystack` (`array`)
- `$strict` (`bool`), default `false`, optional

**Returns**: `mixed`
