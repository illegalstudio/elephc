## array_intersect()

```php
function array_intersect(array $array, ...$arrays): mixed
```

Computes the intersection of arrays.

**Parameters**:
- `$array` (`array`)
- `...$arrays` — variadic: collects excess arguments into `$arrays`.

**Returns**: `mixed`
