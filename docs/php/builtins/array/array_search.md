## array_search()

```php
function array_search(mixed $needle, array $haystack, bool $strict = false): mixed
```

Searches the array for a given value and returns the first corresponding key if successful.

**Parameters**:
- `$needle` (`mixed`)
- `$haystack` (`array`)
- `$strict` (`bool`), default `false`, optional

**Returns**: `mixed`
