## array_push()

```php
function array_push(array $array, ...$values): void
```

Pushes one or more elements onto the end of array.

**Parameters**:
- `$array` (`array`), passed by reference
- `...$values` — variadic: collects excess arguments into `$values`.

**Returns**: `void`
