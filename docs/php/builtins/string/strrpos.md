## strrpos()

```php
function strrpos(string $haystack, string $needle, int $offset = 0): mixed
```

Finds the numeric position of the last occurrence of a substring.

**Parameters**:
- `$haystack` (`string`)
- `$needle` (`string`)
- `$offset` (`int`), default `0`, optional

**Returns**: `mixed`
