## substr_replace()

```php
function substr_replace(string $string, string $replace, int $offset, int $length = null): string
```

Replaces text within a portion of a string.

**Parameters**:
- `$string` (`string`)
- `$replace` (`string`)
- `$offset` (`int`)
- `$length` (`int`), default `null`, optional

**Returns**: `string`
