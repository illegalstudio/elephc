## str_replace()

```php
function str_replace(string $search, string $replace, string $subject, int $count = null): string
```

Replaces all occurrences of a search string with a replacement string.

**Parameters**:
- `$search` (`string`)
- `$replace` (`string`)
- `$subject` (`string`)
- `$count` (`int`), default `null`, optional

**Returns**: `string`
