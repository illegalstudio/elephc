## strpos()

```php
function strpos(string $haystack, string $needle, int $offset = 0): mixed
```

Finds the numeric position of the first occurrence of a substring.

**Parameters**:
- `$haystack` (`string`)
- `$needle` (`string`)
- `$offset` (`int`), default `0`, optional

**Returns**: `mixed`
