## explode()

```php
function explode(string $separator, string $string, int $limit = PHP_INT_MAX): array
```

Splits a string by a separator into an array of substrings.

**Parameters**:
- `$separator` (`string`)
- `$string` (`string`)
- `$limit` (`int`), default `PHP_INT_MAX`, optional

**Returns**: `array`
